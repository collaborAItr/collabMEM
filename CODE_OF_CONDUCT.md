# Code Of Conduct

This collabMEM Open Architecture project is intended to help developers discuss and design user-controlled memory systems for LLM chat apps.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent while staying precise.
- Keep criticism focused on ideas and documentation.
- Respect privacy and intellectual property boundaries.
- Help make the project useful to builders with different technical stacks.

## Unacceptable Behavior

- Harassment or personal attacks.
- Publishing private, sensitive, or proprietary information.
- Pressuring contributors to reveal implementation details they cannot share.
- Misrepresenting speculative ideas as proven science.
- Dismissing user privacy or control concerns.

## Reporting

If a contribution or interaction creates a concern, open a GitHub issue or contact the repository maintainer through the channels listed on the repository profile.

Maintainers may remove comments, close issues, or block contributors who violate these expectations.
